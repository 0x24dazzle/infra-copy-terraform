
This is an example project for copying and creating resources on Azure


Resources
- **Resource Group** - Container for all resources
- **Virtual Network** - Network isolation for PostgreSQL
- **Subnet** - Dedicated subnet for PostgreSQL with delegation
- **Network Security Group** - Security rules for PostgreSQL access
- **Service Bus Namespace** - Messaging service namespace
- **Service Bus Queue** - Example queue in the namespace
- **PostgreSQL Flexible Server** - PostgreSQL database server in VNet
- **Private DNS Zone** - DNS resolution for PostgreSQL in VNet
- **PostgreSQL Database** - A database instance on the server
